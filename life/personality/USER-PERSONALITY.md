## 核心特征

### 沟通风格
- **极简主义** - 能用 5 个字说清楚绝不用 10 个字
- **命令式** - 直接下达指令，不绕弯子
- **零废话** - 没有寒暄、没有 emoji、没有语气词
- **高效** - 一句话包含所有必要信息

### 典型表达模式
```
❌ "你好啊，最近怎么样？"
✅ "hello"

❌ "我想问一下你现在还在线吗？"
✅ "你活了吗"

❌ "我记得你之前好像断了一段时间？"
✅ "你之前断了很久"

❌ "能不能帮我安装一个技能，让我可以有更好的记忆功能？"
✅ "安装技能让你获得记忆"
```

### 决策方式
- **目标导向** - 直接说想要什么结果
- **技术细节明确** - 知道具体要什么（GitHub 链接、技能名称、文件路径）
- **一次性说完** - 把所有要求放在一条消息里
- **要求验证** - "部署完成后立即确认并测试"

### 语言特征
| 特征 | 表现 |
|------|------|
| Emoji 使用 | 从不使用 |
| 标点符号 | 极少，句末常无标点 |
| 句子长度 | 平均 < 20 字（技术指令除外） |
| 语气词 | 无 |
| 敬语/客套 | 无 |

### 高频词汇
- 直接指令动词：安装、实现、配置、更新、测试
- 技术术语：GitHub、skills、cron、脚本、目录结构
- 时间词：立即、同时、完成后
